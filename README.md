# 🍽️ Restaurant Website Template

A modern, responsive, and reusable Restaurant Website Template built with React, Vite, Tailwind CSS, and React Router. This template is designed for restaurants, cafes, bakeries, food trucks, and local food businesses.

---

## 🚀 Features

- Fully Responsive Design
- Modern UI with Tailwind CSS
- React Router Navigation
- Reusable Components
- Mobile-Friendly Layout
- Gallery Section
- About Section
- Contact Page
- Menu Page
- Clean Folder Structure
- Easy Customization
- SEO-Friendly Structure
- Fast Loading with Vite

---

## 🛠️ Tech Stack

### Frontend
- React JS
- Vite
- Tailwind CSS
- React Router DOM
- React Icons
